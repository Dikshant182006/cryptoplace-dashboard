# Cryptoplace

Cryptoplace is a modern crypto dashboard built with React, Tailwind CSS, and CoinGecko API. It displays real-time cryptocurrency market data with a clean and responsive user interface.

## Features

- Real-time crypto market data
- Search functionality
- Responsive UI
- Modern gradient-based design
- Built with React and Tailwind CSS

## Tech Stack

- React
- Tailwind CSS
- JavaScript
- CoinGecko API
- Vite
