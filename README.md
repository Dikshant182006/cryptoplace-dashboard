# Cryptoplace - Crypto Price Dashboard

Cryptoplace is a responsive cryptocurrency price dashboard built with React. It allows users to explore live cryptocurrency prices, view market data, check coin details, and track crypto market trends in a clean modern UI.

## Live Demo

https://cryptoplace-dashboard-mk54.vercel.app/

## Features

- Live cryptocurrency market data
- Search and explore cryptocurrencies
- Coin detail page with dynamic routing
- Price chart using historical market data
- Currency switcher: USD, INR, EUR
- Responsive navbar with mobile menu
- Dark and light theme toggle
- Portfolio tracker page
- Clean and modern UI using Tailwind CSS

## Tech Stack

- React.js
- React Router DOM
- Tailwind CSS
- CoinGecko API
- React Google Charts
- Vercel

## Pages

- Home Page
